# Login Module – Manual Test Cases
This repository contains 25 professional manual test cases for a standard Login Module, including UI, functional, positive, negative, and edge-case scenarios.

# 📌 Project Overview
This project is part of my SDET Internship, where I created structured test cases for:
- Email Field
- Password Field
- Login Button
- Error Messages
- UI Checks
- Edge Cases

# 📂 Files Included
**File**	                            **Description**
  Login_Module_TestCases.xlsx	        Complete Excel file containing 25 test cases
  /screenshots	                      Contains supporting screenshots

# 🧪 Test Case Format
Each test case includes:
- TC_ID
- Test Scenario
- Test Description
- Test Steps
- Expected Result
